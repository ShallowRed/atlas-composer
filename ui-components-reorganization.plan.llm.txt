# UI Components Reorganization - Implementation Plan

## Objective
Reorganize the ui components folder by creating subdirectories with separated concerns to improve maintainability and discoverability.

## Affected Domains
- [ ] Vue Architecture (docs/vue-architecture.llm.txt) - Component structure and organization

## Context
- Architecture: docs/architecture.llm.txt
- Vue Layer: docs/vue-architecture.llm.txt
- Current: 24 components in flat structure (src/components/ui/)
- Already exists: src/components/ui/projections/ subfolder

## Proposed Structure

```
src/components/ui/
├── layout/              # Layout and page structure
│   ├── AppHeader.vue
│   ├── AppFooter.vue
│   ├── MainLayout.vue
│   └── ScrollableContent.vue
├── primitives/          # Core reusable UI elements
│   ├── AccordionItem.vue
│   ├── Alert.vue
│   ├── ButtonGroup.vue
│   ├── CardContainer.vue
│   ├── LabelWithIcon.vue
│   └── Modal.vue
├── forms/               # Form controls and inputs
│   ├── CheckboxControl.vue
│   ├── RangeSlider.vue
│   ├── SelectControl.vue
│   └── ToggleControl.vue
├── settings/            # Settings and preferences
│   ├── LanguageSelector.vue
│   └── ThemeSelector.vue
├── projections/         # Projection-related components (already exists)
│   ├── ProjectionConfirmDialog.vue
│   ├── ProjectionInfo.vue
│   ├── ProjectionParamsControls.vue
│   ├── ProjectionSelector.vue
│   ├── ProjectionDropdown.vue
│   ├── ProjectionRecommendationBadge.vue
│   └── ProjectionSearchBar.vue
├── import/              # Import-related components
│   ├── ImportControls.vue
│   └── ImportModal.vue
└── export/              # Export-related components
    ├── CompositeExportDialog.vue
    └── ToastNotification.vue
```
