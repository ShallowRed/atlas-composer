# Préparation Grand Prix (2.Nc3 & 3.Bb5) - Blancs vs Sicilienne (2000 Elo)

## 1. L'Arme Principale : Le Système Tiviakov (3.Bb5)

Cette ligne est votre priorité pour le week-end. Elle vise à infliger une faiblesse structurelle (pions doublés) aux Noirs tout en gardant une position saine, limitant le contre-jeu tactique adverse.

### Partie Modèle : La domination positionnelle

Cette partie de Gawain Jones est la référence absolue. Observez comment les Blancs ferment le centre, fixent les faiblesses, et transfèrent l'attaque à l'aile roi une fois l'aile dame bloquée.

```pgn
[PlyCount "53"]

1. e4 c5 2. Nc3 Nc6 3. Bb5 g6 4. Bxc6 {Le coup clé. On ne recule pas, on crée la faiblesse structurelle immédiatement.} dxc6 5. f4 Bg7 6. d3 Nf6 7. Nf3 O-O 8. O-O b6 9. Qe1 {Début du plan standard : la dame se dirige vers h4.} Bg4 10. Ne5 Qc7 11. Qh4 Be6 12. Nf3 {Le cavalier se replace pour soutenir f5.} h6 13. f5! {La rupture thématique. Les Blancs ouvrent la colonne f.} g5 14. Nxg5 hxg5 15. Bxg5 Bc8 16. Rf3 Rd8 17. Bh6 Ne8 18. Bxg7 Nxg7 19. Rh3 f6 20. Qh7+ Kf7 21. Qg6+ Kg8 22. Rf1 e6 23. Qh7+ Kf7 24. fxe6+ Bxe6 25. Rh6 Qe5 26. Qg6+ Kg8 27. Rxf6 1-0
```

### Théorie : Réponse au coup actif 3...Nd4

Si les Noirs refusent les pions doublés en jouant le cavalier, les Blancs jouent pour le développement rapide.

```pgn

[Annotator "Préparation"]

1. e4 c5 2. Nc3 Nc6 3. Bb5 Nd4 4. Nf3 {Développement prioritaire. On laisse les Noirs prendre la paire de fous en échange de temps.} Nxb5 (4... Bg7 5. Nxd4 cxd4 6. Ne2 a6 7. Bd3 {Le fou se recycle bizarrement mais solidement pour soutenir e4/f4}) 5. Nxb5 a6 6. Nc3 b5 7. d4 {Transition vers une Sicilienne ouverte favorable ou d3 pour rester fermé.} cxd4 8. Qxd4 Bb7 9. Bg5 {Clouage gênant pour le développement noir.} *
```

## 2. Le Plan d'Attaque "Caveman" (Grand Prix Classique)

À utiliser si les Noirs jouent passivement (par exemple 2...d6) ou si vous avez besoin de compliquer la position. C'est le plan "primitif" mais très dangereux : Dame en h4, Fou en h6, Pion en f5.

### Partie Modèle : L'attaque de mat directe

Jones démontre ici la violence de la rupture f5, même au prix d'un pion.

```pgn
[Event "European Club Cup"]

[PlyCount "37"]

1. e4 c5 2. Nc3 d6 3. f4 g6 4. Nf3 Bg7 5. Bc4 Nc6 6. O-O e6 7. d3 Nge7 {Le setup défensif le plus solide des Noirs.} 8. Qe1 a6 9. f5! {Le sacrifice thématique pour ouvrir les lignes avant que les Noirs ne roquent ou ne consolident.} gxf5 10. Qh4 fxe4 11. Ng5! {Cible f7 avec violence.} exd3 12. Rxf6! {Sacrifice de qualité pour détruire le défenseur des cases noires.} Bxf6 13. Qh7 Rf8 14. Nce4 Bd4+ 15. Kh1 d5 16. Nxf7 Rxf7 17. Qg8+ Rf8 18. Nd6+ Kd7 19. Qg7+ 1-0
```

## 3. Gestion des Réponses "Anti-Grand Prix" (2...e6)

C'est le talon d'Achille du joueur de Grand Prix. Si les Noirs jouent 2...e6 pour préparer...d5 rapidement, ne jouez pas 3.f4 (qui permet le Gambit Tal). Transposez dans une ligne hybride solide.

### Répertoire : La structure Rossolimo/Française

L'objectif est d'empêcher les Noirs de liquider le centre trop facilement.

```pgn
[Annotator "Préparation"]

1. e4 c5 2. Nc3 e6 {L'ordre de coups qui tente d'éviter le Grand Prix.} 3. Nf3! {Flexibilité. Ne jouez pas f4 ici.} d5 4. Bb5+! {Force les Noirs à bloquer avec une pièce, ce qui ralentit leur expansion.} Bd7 (4... Nc6 5. exd5 exd5 6. O-O {Simple et sain. Les Blancs jouent contre le pion isolé d5 ou pour l'attaque sur la colonne e.}) 5. Bxd7+ Qxd7 6. exd5 exd5 7. d4! {On transpose dans une sorte de Française d'échange ou de Gambit Dame où les Blancs sont très confortables et les Noirs n'ont pas leur dynamique habituelle.} *
```

## 4. Anti-Fianchetto (L'étau Maroczy Hybride)

Si l'adversaire joue un Dragon Accéléré (2...g6), le système 3.f4 reste excellent, mais le placement du Fou en b5 est crucial pour gêner.

```pgn
[Annotator "Préparation"]

1. e4 c5 2. Nc3 g6 3. f4 Bg7 4. Nf3 Nc6 5. Bb5! {Toujours ce coup. Si 5.Bc4, les Noirs égalisent souvent.} Nd4 6. O-O a6 (6... Nxb5 7. Nxb5 d6 8. d3 Nf6 9. Qe1 O-O 10. Qh4 {Retour au plan d'attaque standard.}) 7. Bd3! {Une subtilité moderne. Le fou protège e4, soutient f5 à distance et se recycle plus tard.} d6 8. Nxd4 cxd4 9. Ne2 *
```
